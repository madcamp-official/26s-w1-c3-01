# MUK PICK Backend

API baseline scaffold for the menu recommendation service.

## Stack

- Node.js
- Express
- TypeScript

## Scripts

```bash
npm install
npm run dev
```

Default API prefix: `/api/v1`

## Structure

```text
src/
  app.ts
  server.ts
  config/
  controllers/
  middleware/
  routes/
  services/
  types/
  utils/
```
