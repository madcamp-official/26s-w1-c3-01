## 기능 명세서

구현할 기능은 사용자가 개인 또는 집단 식사 상황에서 메뉴를 효율적으로 결정할 수 있도록 지원하는 기능으로 구성한다.

필수 기능은 사용자 선호도와 알러지 정보를 기반으로 메뉴 후보를 필터링하고, 식사 시간대와 참여자들의 선호도를 반영하여 메뉴를 랭킹 형태로 추천하는 기능이다.
또한 집단 약속 생성, 참여자 정보 취합, 추천 결과 및 추천 이유 제공 기능을 포함한다.

선택 기능은 추천 메뉴에 맞는 주변 식당 정보 제공, 날짜와 시간에 따른 선호도 분석, 다이어트·웰니스 등 개인 식단 목표 기반 추천 기능으로 구성한다.

### 기능 범위 요약

### 필수 기능

- 사용자 정보 관리
- 개인 음식 선호도 분석
- 메뉴 랭킹 추천
- 집단 약속 메뉴 추천
- 식사 히스토리 기반 추천
- 알러지 및 제한 조건 필터링

### 선택 기능

- 주변 식당 추천
- 날짜 및 시간에 따른 메뉴 선호도 반영
- 개인 식단 목표 기반 추천
- 새로운 메뉴 시도 추천
